在计算机科学中，数据结构是程序设计的基础，而栈和队列作为两种基本且重要的线性数据结构，在软件开发和算法设计中扮演着关键角色。本文将从**定义、特点、实现方式、应用场景、时间复杂度**等多个维度，全面解析栈和队列这两种核心数据结构。

# 数据结构栈与队列：LIFO与FIFO核心机制深度解析

## 一、数据结构概览

数据结构是计算机存储、组织数据的方式，栈和队列都是线性数据结构的典型代表：

- **栈（Stack）**：后进先出（LIFO - Last In First Out）的数据结构
- **队列（Queue）**：先进先出（FIFO - First In First Out）的数据结构

这两种数据结构虽然都是线性结构，但其操作规则和应用场景截然不同。

## 二、栈（Stack）详解

### 2.1 定义

栈是一种受限的线性表，只允许在表的一端进行插入和删除操作，这一端被称为栈顶（top），另一端称为栈底（bottom）。栈遵循后进先出（LIFO）的原则，最后插入的元素最先被删除。

### 2.2 核心操作

- **push(item)**：将元素压入栈顶
- **pop()**：弹出栈顶元素
- **peek()/top()**：查看栈顶元素但不删除
- **isEmpty()**：判断栈是否为空
- **size()**：获取栈中元素数量

### 2.3 代码实现

```java
import java.util.EmptyStackException;

/**
 * 栈的数组实现
 */
class ArrayStack<T> {
    private Object[] data;      // 存储栈元素的数组
    private int top;            // 栈顶指针
    private int capacity;       // 栈容量
    
    // 构造函数
    public ArrayStack(int capacity) {
        this.capacity = capacity;
        this.data = new Object[capacity];
        this.top = -1;  // 栈为空时，栈顶指针为-1
    }
    
    public ArrayStack() {
        this(10);  // 默认容量为10
    }
    
    /**
     * 入栈操作
     */
    public void push(T item) {
        if (isFull()) {
            throw new StackOverflowError("栈已满，无法添加元素");
        }
        data[++top] = item;
    }
    
    /**
     * 出栈操作
     */
    @SuppressWarnings("unchecked")
    public T pop() {
        if (isEmpty()) {
            throw new EmptyStackException();
        }
        T item = (T) data[top];
        data[top--] = null;  // 避免内存泄漏
        return item;
    }
    
    /**
     * 查看栈顶元素
     */
    @SuppressWarnings("unchecked")
    public T peek() {
        if (isEmpty()) {
            throw new EmptyStackException();
        }
        return (T) data[top];
    }
    
    /**
     * 判断栈是否为空
     */
    public boolean isEmpty() {
        return top == -1;
    }
    
    /**
     * 判断栈是否已满
     */
    public boolean isFull() {
        return top == capacity - 1;
    }
    
    /**
     * 获取栈中元素数量
     */
    public int size() {
        return top + 1;
    }
    
    /**
     * 清空栈
     */
    public void clear() {
        for (int i = 0; i <= top; i++) {
            data[i] = null;
        }
        top = -1;
    }
}

/**
 * 栈的链表实现
 */
class LinkedStack<T> {
    private Node<T> top;    // 栈顶节点
    private int size;       // 栈大小
    
    // 节点内部类
    private static class Node<T> {
        T data;
        Node<T> next;
        
        Node(T data) {
            this.data = data;
        }
    }
    
    public LinkedStack() {
        this.top = null;
        this.size = 0;
    }
    
    /**
     * 入栈操作
     */
    public void push(T item) {
        Node<T> newNode = new Node<>(item);
        newNode.next = top;
        top = newNode;
        size++;
    }
    
    /**
     * 出栈操作
     */
    public T pop() {
        if (isEmpty()) {
            throw new EmptyStackException();
        }
        T data = top.data;
        top = top.next;
        size--;
        return data;
    }
    
    /**
     * 查看栈顶元素
     */
    public T peek() {
        if (isEmpty()) {
            throw new EmptyStackException();
        }
        return top.data;
    }
    
    /**
     * 判断栈是否为空
     */
    public boolean isEmpty() {
        return top == null;
    }
    
    /**
     * 获取栈大小
     */
    public int size() {
        return size;
    }
    
    /**
     * 清空栈
     */
    public void clear() {
        top = null;
        size = 0;
    }
}

/**
 * 栈操作演示
 */
public class StackDemo {
    public static void main(String[] args) {
        // 使用数组实现的栈
        ArrayStack<Integer> arrayStack = new ArrayStack<>(5);
        
        // 入栈操作
        arrayStack.push(1);
        arrayStack.push(2);
        arrayStack.push(3);
        
        System.out.println("栈大小：" + arrayStack.size()); // 3
        System.out.println("栈顶元素：" + arrayStack.peek()); // 3
        
        // 出栈操作
        System.out.println("出栈：" + arrayStack.pop()); // 3
        System.out.println("出栈：" + arrayStack.pop()); // 2
        System.out.println("栈大小：" + arrayStack.size()); // 1
        
        // 使用链表实现的栈
        LinkedStack<String> linkedStack = new LinkedStack<>();
        linkedStack.push("A");
        linkedStack.push("B");
        linkedStack.push("C");
        
        System.out.println("\n链表栈演示：");
        while (!linkedStack.isEmpty()) {
            System.out.println("出栈：" + linkedStack.pop());
        }
    }
}
```

- **pop()**: O(1)

6. **深度优先搜索（DFS）**：图和树的遍历算法

## 三、队列（Queue）详解

### 3.1 定义

队列是一种受限的线性表，只允许在表的一端进行插入操作（入队），在另一端进行删除操作（出队）。队列遵循先进先出（FIFO）的原则，最先插入的元素最先被删除。

### 3.2 核心操作

- **enqueue(item)**：将元素添加到队尾
- **dequeue()**：从队头移除元素
- **front()/peek()**：查看队头元素但不删除
- **isEmpty()**：判断队列是否为空
- **size()**：获取队列中元素数量

### 3.3 代码实现

```java
import java.util.NoSuchElementException;

/**
 * 队列的数组实现（循环队列）
 */
class ArrayQueue<T> {
    private Object[] data;      // 存储队列元素的数组
    private int front;          // 队头指针
    private int rear;           // 队尾指针
    private int size;           // 队列大小
    private int capacity;       // 队列容量
    
    public ArrayQueue(int capacity) {
        this.capacity = capacity;
        this.data = new Object[capacity];
        this.front = 0;
        this.rear = -1;
        this.size = 0;
    }
    
    public ArrayQueue() {
        this(10);  // 默认容量为10
    }
    
    /**
     * 入队操作
     */
    public void enqueue(T item) {
        if (isFull()) {
            throw new IllegalStateException("队列已满");
        }
        rear = (rear + 1) % capacity;  // 循环操作
        data[rear] = item;
        size++;
    }
    
    /**
     * 出队操作
     */
    @SuppressWarnings("unchecked")
    public T dequeue() {
        if (isEmpty()) {
            throw new NoSuchElementException("队列为空");
        }
        T item = (T) data[front];
        data[front] = null;  // 避免内存泄漏
        front = (front + 1) % capacity;  // 循环操作
        size--;
        return item;
    }
    
    /**
     * 查看队头元素
     */
    @SuppressWarnings("unchecked")
    public T front() {
        if (isEmpty()) {
            throw new NoSuchElementException("队列为空");
        }
        return (T) data[front];
    }
    
    /**
     * 判断队列是否为空
     */
    public boolean isEmpty() {
        return size == 0;
    }
    
    /**
     * 判断队列是否已满
     */
    public boolean isFull() {
        return size == capacity;
    }
    
    /**
     * 获取队列大小
     */
    public int size() {
        return size;
    }
    
    /**
     * 清空队列
     */
    public void clear() {
        for (int i = 0; i < capacity; i++) {
            data[i] = null;
        }
        front = 0;
        rear = -1;
        size = 0;
    }
}

/**
 * 队列的链表实现
 */
class LinkedQueue<T> {
    private Node<T> front;    // 队头节点
    private Node<T> rear;     // 队尾节点
    private int size;         // 队列大小
    
    // 节点内部类
    private static class Node<T> {
        T data;
        Node<T> next;
        
        Node(T data) {
            this.data = data;
        }
    }
    
    public LinkedQueue() {
        this.front = null;
        this.rear = null;
        this.size = 0;
    }
    
    /**
     * 入队操作
     */
    public void enqueue(T item) {
        Node<T> newNode = new Node<>(item);
        if (isEmpty()) {
            front = rear = newNode;
        } else {
            rear.next = newNode;
            rear = newNode;
        }
        size++;
    }
    
    /**
     * 出队操作
     */
    public T dequeue() {
        if (isEmpty()) {
            throw new NoSuchElementException("队列为空");
        }
        T data = front.data;
        front = front.next;
        if (front == null) {  // 队列变空
            rear = null;
        }
        size--;
        return data;
    }
    
    /**
     * 查看队头元素
     */
    public T front() {
        if (isEmpty()) {
            throw new NoSuchElementException("队列为空");
        }
        return front.data;
    }
    
    /**
     * 判断队列是否为空
     */
    public boolean isEmpty() {
        return front == null;
    }
    
    /**
     * 获取队列大小
     */
    public int size() {
        return size;
    }
    
    /**
     * 清空队列
     */
    public void clear() {
        front = rear = null;
        size = 0;
    }
}

/**
 * 队列操作演示
 */
public class QueueDemo {
    public static void main(String[] args) {
        // 使用数组实现的队列
        ArrayQueue<String> arrayQueue = new ArrayQueue<>(5);
        
        // 入队操作
        arrayQueue.enqueue("A");
        arrayQueue.enqueue("B");
        arrayQueue.enqueue("C");
        
        System.out.println("队列大小：" + arrayQueue.size()); // 3
        System.out.println("队头元素：" + arrayQueue.front()); // A
        
        // 出队操作
        System.out.println("出队：" + arrayQueue.dequeue()); // A
        System.out.println("出队：" + arrayQueue.dequeue()); // B
        System.out.println("队列大小：" + arrayQueue.size()); // 1
        
        // 使用链表实现的队列
        LinkedQueue<Integer> linkedQueue = new LinkedQueue<>();
        linkedQueue.enqueue(10);
        linkedQueue.enqueue(20);
        linkedQueue.enqueue(30);
        
        System.out.println("\n链表队列演示：");
        while (!linkedQueue.isEmpty()) {
            System.out.println("出队：" + linkedQueue.dequeue());
        }
    }
}
```

## 四、栈与队列的对比

| 特性 | 栈（Stack） | 队列（Queue） |
|------|-------------|---------------|
| 存取原则 | LIFO（后进先出） | FIFO（先进先出） |
| 操作端 | 只在栈顶操作 | 在两端操作（队头出队，队尾入队） |
| 主要操作 | push, pop, peek | enqueue, dequeue, front |
| 应用场景 | 函数调用、表达式求值 | 任务调度、BFS |
| 时间复杂度 | 所有操作O(1) | 所有操作O(1) |

## 五、特殊变体

### 5.1 双端队列（Deque）

双端队列允许在两端进行插入和删除操作，是栈和队列的结合。

```java
import java.util.ArrayDeque;
import java.util.Deque;

public class DequeDemo {
    public static void main(String[] args) {
        Deque<Integer> deque = new ArrayDeque<>();
        
        // 从队头插入（类似栈的push）
        deque.push(1);
        deque.push(2);
        System.out.println("从队头插入后：" + deque); // [2, 1]
        
        // 从队尾插入（类似队列的enqueue）
        deque.addLast(3);
        System.out.println("从队尾插入后：" + deque); // [2, 1, 3]
        
        // 从队头删除（类似栈的pop）
        System.out.println("从队头删除：" + deque.pop()); // 2
        
        // 从队尾删除
        System.out.println("从队尾删除：" + deque.removeLast()); // 3
        System.out.println("最终结果：" + deque); // [1]
    }
}
```

### 5.2 优先队列（PriorityQueue）

优先队列中元素按优先级排序，优先级最高的元素先出队。

```java
import java.util.PriorityQueue;

public class PriorityQueueDemo {
    public static void main(String[] args) {
        // 最小堆（默认）
        PriorityQueue<Integer> minHeap = new PriorityQueue<>();
        minHeap.offer(5);
        minHeap.offer(2);
        minHeap.offer(8);
        minHeap.offer(1);
        
        System.out.println("优先队列内容：");
        while (!minHeap.isEmpty()) {
            System.out.print(minHeap.poll() + " "); // 输出：1 2 5 8
        }
        
        // 最大堆（使用自定义比较器）
        PriorityQueue<Integer> maxHeap = new PriorityQueue<>((a, b) -> b - a);
        maxHeap.offer(5);
        maxHeap.offer(2);
        maxHeap.offer(8);
        maxHeap.offer(1);
        
        System.out.println("\n最大堆优先队列：");
        while (!maxHeap.isEmpty()) {
            System.out.print(maxHeap.poll() + " "); // 输出：8 5 2 1
        }
    }
}
```

## 六、实际应用案例

### 6.1 用栈实现表达式求值

```java
public class ExpressionEvaluator {
    /**
     * 计算后缀表达式（逆波兰表示法）
     */
    public static int evaluatePostfix(String expression) {
        Stack<Integer> stack = new Stack<>();
        String[] tokens = expression.split("\\s+");
        
        for (String token : tokens) {
            if (isOperator(token)) {
                int b = stack.pop();
                int a = stack.pop();
                int result = performOperation(a, b, token);
                stack.push(result);
            } else {
                stack.push(Integer.parseInt(token));
            }
        }
        
        return stack.pop();
    }
    
    private static boolean isOperator(String token) {
        return token.equals("+") || token.equals("-") || 
               token.equals("*") || token.equals("/");
    }
    
    private static int performOperation(int a, int b, String operator) {
        switch (operator) {
            case "+": return a + b;
            case "-": return a - b;
            case "*": return a * b;
            case "/": return a / b;
            default: throw new IllegalArgumentException("未知操作符: " + operator);
        }
    }
    
    public static void main(String[] args) {
        // 计算 2 3 + 4 * = (2 + 3) * 4 = 20
        String postfix = "2 3 + 4 *";
        System.out.println("后缀表达式 " + postfix + " 的结果: " + evaluatePostfix(postfix));
    }
}
```

### 6.2 用队列实现BFS算法

```java
import java.util.*;

public class BFSExample {
    static class Graph {
        private Map<Integer, List<Integer>> adjacencyList;
        
        public Graph() {
            adjacencyList = new HashMap<>();
        }
        
        public void addEdge(int source, int destination) {
            adjacencyList.computeIfAbsent(source, k -> new ArrayList<>()).add(destination);
            adjacencyList.computeIfAbsent(destination, k -> new ArrayList<>()).add(source);
        }
        
        public void BFS(int startVertex) {
            Set<Integer> visited = new HashSet<>();
            Queue<Integer> queue = new LinkedList<>();
            
            visited.add(startVertex);
            queue.offer(startVertex);
            
            System.out.println("BFS遍历结果：");
            while (!queue.isEmpty()) {
                int currentVertex = queue.poll();
                System.out.print(currentVertex + " ");
                
                // 访问当前顶点的所有邻居
                List<Integer> neighbors = adjacencyList.get(currentVertex);
                if (neighbors != null) {
                    for (int neighbor : neighbors) {
                        if (!visited.contains(neighbor)) {
                            visited.add(neighbor);
                            queue.offer(neighbor);
                        }
                    }
                }
            }
            System.out.println();
        }
    }
    
    public static void main(String[] args) {
        Graph graph = new Graph();
        graph.addEdge(0, 1);
        graph.addEdge(0, 2);
        graph.addEdge(1, 3);
        graph.addEdge(1, 4);
        graph.addEdge(2, 5);
        
        graph.BFS(0); // 从顶点0开始BFS遍历
    }
}
```

## 七、性能优化建议

1. **选择合适的实现方式**：
   - 数组实现：适合大小固定且已知的场景
   - 链表实现：适合大小动态变化的场景

2. **内存管理**：
   - 及时清理已删除的元素引用，避免内存泄漏
   - 使用泛型避免装箱拆箱开销

3. **容量预设**：
   - 对于数组实现，预设合适的初始容量避免频繁扩容
   - 对于链表实现，注意链表过长时的性能影响

## 八、总结

栈和队列是两种基本而重要的数据结构：

- **栈**：后进先出（LIFO）的特性使其适用于递归模拟、表达式求值等场景
- **队列**：先进先出（FIFO）的特性使其适用于任务调度、广度优先搜索等场景

在实际开发中，应根据具体需求选择合适的数据结构和实现方式。Java标准库中提供了`Stack`、`Queue`、`Deque`等接口及其实现类，但在实际开发中，通常推荐使用`ArrayDeque`作为栈和队列的实现，因为它的性能优于传统的`Stack`类。

掌握栈和队列的原理和实现，对于理解更复杂的数据结构和算法具有重要意义，是算法设计和程序开发的基石。
